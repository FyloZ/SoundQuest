# SoundQuest
## Comment build le projet dans Eclipse
- Importer le projet grâce avec Gradle.
- Définir la "compliance du projet" en Java SE 1.7 au minimum.
- La classe principale est DesktopLauncher dans net.fyloz.soundquest.destop .
- Pour build le projet, créer une configuration Java Application.
- Classe principale: net.fyloz.soundquest.desktop.DesktopLauncher
- Dans arguments, définir le Working Directory à core\assets
- Tout est prêt!

## Contrôles:
- A/Droite: Aller à droite.
- D/Gauche: Aller à gauche.
- Espace: Sauter.
